Кто такой java разработчик

Java разработчик это человек умеющий решать большую часть задач из мира IT: работать с данными, создавать приложения для настольных компьютеров, мобильных телефонов, вэб серверов и так далее.

На первых шагах java выглядит более сложным языком чем Python или Javascript, но после пары недель погружения, вы поймете, что за этой сложностью скрывается очень удобный механизм написания кода.

Поняв принципы java, вы вряд ли захотите писать на других языках.

Модуль 1

Java core

Краткое описание: погружение в самые основы языка
  • Настроим свой компьютер для написания кода на java
  • Как отправлять код, который вы написали, в интернет, чтобы другие люди могли его видеть.
  • Что такое типы данных, и для чего они нужны, как с ними работать
  • Что такое массивы, как работать с ними
  • Узнаем основные принципы Объектно-ориентированного программирования: абстракция, инкапсуляция, наследование, полиморфизм.
  • Узнаем что такое класс, атрибуты, методы, конструктор, геттеры, сеттеры. Научимся понимать разницу между классом и его экземплярами.
  • Поймем зачем нам нужны абстрактные классы, интерфейсы, вложенны

Модуль 3

Углубление в git, maven, lombok, введение в тестирование

Краткое описание: как собирать и тестировать свои проекты, чтобы ими могли пользоваться другие разработчики.

  • Познакомимся с системой сборки maven, поймем для чего он нужен
  • Научимся создавать свои maven проекты, добавлять зависимости разных типов
  • Изучим основные стадии сборки приложения
  • Поработаем с первой сторонней зависимостью lombook, и узнаем как ускорить написание кода с его помощью
  • Погрузимся в тестирование: поймем для чего оно нужно, какие типы тестирования бывают
  • Напишем свои тесты с помощью библиотек junit, testng.
  • Поймем для чего нужны Mock, Stub, Spy

Модуль 4

Sql, jdbc, hibernate

Краткое описание: научимся работать с базами данных используя PostgreSQL
  • Узнаем что такое база данных, реляционная модель хранения данных, нормальные формы
  • Узнаем о языке работы с базами данных SQL, и начнем писать на нем запросы
  • Познакомимся с такими понятиями как primary key, foreign key, индексы, последовательности
  • Начнем работать с базой данных из java отправляя sql запросы через jdbc
  • Освоим транзакции, их уровни и как их можно отменять
  • Поймем что такое ORM, и как он упрощает жизнь разработчику
  • Подключим hibernate, опишем наши данные из базы данных с точки зрения java классов.
  • Изучим внутренние механизмы hibernate.

Модуль 5

Spring core + boot + web + rest + data

Краткое описание: делаем вэб приложение с помощью Spring!
  • Узнаем что такое IOC контейнер, напишем свою реализацию этого контейнера.
  • Познакомимся со фрейморком Spring, посмотрим как он реализует Dependency injection
  • Научимся создавать свои аннотации для Spring
  • Поднимем свой веб сервер с помощью Spring boot
  • Расскажу про протокол http, какие виды http запросов бывают.
  • Научимся обрабатывать http запросы с помощью Spring web
  • На практике поработаем с json объектами и научимся их генерировать с помощью Spring rest
  • Поработаем с базами данных с помощью Spring Data
